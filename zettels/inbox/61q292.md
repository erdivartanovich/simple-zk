# Bash subshell with parenthesis

Source: https://www.youtube.com/watch?v=Sx9zG7wa4FA
Author: Dave Eddy

```bash
# notice in this function we are using parenthesis as the fucntion block
# this make every code in the greet() is executed in a subshell/ local scope

greet() (
  name=$1
  echo "Hello $name"
)

goodbye() {
  name=$1
  echo "Bye $name"
}
```
